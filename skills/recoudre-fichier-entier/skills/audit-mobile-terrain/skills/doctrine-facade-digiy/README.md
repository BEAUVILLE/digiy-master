# DIGIY Master

Plugin Codex pour DIGIYLYFE.

## Contenu
- recoudre-fichier-entier
- audit-mobile-terrain
- review-pr-digiy
- doctrine-facade-digiy

## But
Donner à Codex le moule maître DIGIY :
- front humain,
- respect du rail réel,
- priorité mobile,
- fichier entier,
- zéro démo inutile.

## Usage
Installe le plugin, puis invoque la skill voulue ou laisse Codex la choisir si la description correspond à la tâche.

## Exemple de prompts
- Utilise recoudre-fichier-entier pour corriger cockpit.html sans toucher au backend.
- Utilise audit-mobile-terrain pour vérifier planning.html sur téléphone.
- Utilise review-pr-digiy pour relire cette PR avant merge.
- Utilise doctrine-facade-digiy pour humaniser les boutons de la page publique.
